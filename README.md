# Spotify Lyric: Traditional To Simplified Chinese
A tool that automatically converts traditional lyrics on Spotify to simplified.

# REQUIREMENTS
1. MS build tools with C++ development added
   - download at https://visualstudio.microsoft.com/visual-cpp-build-tools/
2. pip install all the python libraries in the requirements.txt
   - you can download requirements.txt
   - then cd to the place you downloaded then run the command below
   - pip install -r requirements.txt

# USEAGE
Either run in terminal with calling python (make sure python is added to PATH) or run lyric_translator.py natively in an editor (VS code)
